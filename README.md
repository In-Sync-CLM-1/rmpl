# RMPL CRM

Healthcare Recruitment Platform - Streamline your staffing workflow.

## Tech Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Supabase (backend)

## Development

```sh
# Install dependencies
npm i

# Start the development server
npm run dev
```

## Deployment

The app is deployed via Azure Static Web Apps. Pushing to `main` triggers automatic build and deploy.
